# ブックマークアプリ Phase4以降の構想メモ

## Phase 3.5

レイアウトの調整をしたい

- ログイン画面では左側の再度メニューは不要
- ブックマーク一覧画面では、ページタイトルの下にURLを表示する
- ブックマーク一覧画面では、ページタイトルまたはURLをクリックしたら該当のページを新規タブで表示する
- ブックマーク詳細画面にURLが表示されていないので、URLを表示する
- ブックマーク登録画面のスマホ表示の時、「追加」ボタンに「＋」ボタンが重なっている。追加画面で新たに「＋」ボタンは不要のため、「＋」ボタンを非表示とする
- スマホ表示のブックマーク登録画面のヘッダーにも「＋」ボタンがあるがこちらも使わないので削除する

## Phase 4以降

- ブックマーク詳細画面では、記事内容の要約機能を実装する
- 要約する機能では、該当URLをスクレイピングして内容を取得し、ClaudeのAPIを利用して要約したテキストを表示する
- ブックマーク一覧ページ、ブックマーク詳細ページのURLをクリックして新規タブで該当のページを開いたら、ブックマークのステータスを既読にする
