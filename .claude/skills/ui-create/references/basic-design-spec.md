# 基本デザイン仕様

個人開発アプリのUI設計に共通して適用するベース仕様です。

---

## 配色

### ベースカラー
- 背景色: #ffffff
- 本文テキスト: #7f7f7f
- 見出し（H2, H3）: #696969
- key color（visumo pink）: #EDAA86
- bold color（pink）: #E0765F

### テキスト用途別
- 画面タイトル（H1）: key color #EDAA86
- セクション見出し（H2, H3）: #696969
- 本文: #7f7f7f

### アクセントカラー
- cyan: #65B6D4 → リンク、情報バッジ
- purple: #AC7AB3 → タグ、カテゴリ
- green: #7CC7B7 → 成功メッセージ、完了状態
- yellow: #F5DC73 → 警告

### ステータスカラー
- エラー / 重要バッジ: bold color #E0765F

---

## フォント
- 日本語: Noto Sans JP（Google Fonts）
- 英数字: Inter（Google Fonts）
- フォントスタック: `'Inter', 'Noto Sans JP', sans-serif`
- ベースサイズ: 16px

### フォントサイズスケール
- xs: 12px
- sm: 14px
- md: 16px（本文）
- lg: 18px
- xl: 20px
- 2xl: 24px
- 3xl: 30px
- 4xl: 36px

---

## レイアウト
- 対応デバイス: モバイル・デスクトップ両対応（レスポンシブ）
- コンテンツ最大幅: 1280px（中央寄せ）
- 基本余白（padding）: モバイル 16px / デスクトップ 32px

### ブレークポイント
- sm: 640px（スマホ横向き〜）
- md: 768px（タブレット〜）
- lg: 1024px（デスクトップ〜）
- xl: 1280px（ワイドデスクトップ〜）

---

## スペーシング・形状

### スペーシングスケール
4px / 8px / 12px / 16px / 24px / 32px / 48px / 64px

### 角丸（border-radius）
- sm: 4px（バッジなど小さい要素）
- md: 8px（ボタン、入力フォームなど）
- lg: 12px（カード、モーダルなど）
- xl: 16px（大きいカード）
- full: 9999px（pill ボタン、アバターなど）

### シャドウ・ボーダー
- スタイル: フラット（影なし）
- 区切り: ボーダーラインで要素を区切る
- ボーダーカラー: #e5e7eb（薄いグレー）

---

## コンポーネント

### ボタン
| バリアント | 背景 | テキスト | 用途 |
|---|---|---|---|
| Primary | #EDAA86 | #ffffff | 主要アクション |
| Secondary | #e5e7eb | #000000 | サブアクション |
| Danger | #E0765F | #ffffff | 削除・破壊的操作 |

- 角丸: md（8px）
- 影: なし（フラット）

### フォーム
- スタイル: アウトライン（ボーダーで囲む）
- ボーダー（通常）: #e5e7eb
- ボーダー（フォーカス）: key color #EDAA86
- ボーダー（エラー）: bold color #E0765F
- 背景: #ffffff
- 角丸: md（8px）
- ラベル: sm（14px）/ #7f7f7f

---

## ナビゲーション
- 構造: サイドバー
  - デスクトップ: 左固定サイドバー（幅 240px）
  - モバイル: ハンバーガーメニュー → ドロワー表示
- サイドバー背景: #ffffff（右ボーダーで本文エリアと区切る）

---

## アイコン
- セット: Lucide（`lucide-react`）
- サイズ: 16px（sm）/ 20px（md）/ 24px（lg）
- カラー: 基本は本文テキスト色（#7f7f7f）、用途に応じてアクセントカラーを使用
